# Roadmap

## Phase 1: Product shell

- Studio dashboard
- 3D viewport
- Generation queue
- Prompt planner
- Asset cards
- Mock generation pipeline

## Phase 2: Real generation

- Connect public LLM router
- Add text-to-3D adapter
- Add image-to-3D adapter
- Store assets in object storage
- Add job progress events

## Phase 3: Production workflow

- UV unwrap
- PBR texture generation
- Retopology
- Decimation
- Export presets for Unity, Unreal, Godot, WebAR, Shopify, and 3D printing

## Phase 4: Agentic editing

- Scene-aware assistant
- Object selection context
- Modify mesh by language
- Material variations
- Style transfer
- Batch jobs

## Phase 5: Business layer

- Auth
- Teams
- Credits
- Billing
- API keys
- Marketplace
